# RoomieSpend Invitation System - User Guide

## How to Invite Friends to a Group

### Method 1: Show QR Code (Recommended for In-Person)

1. Open your group in RoomieSpend
2. Tap the "Invite Friends" card (appears when group has no expenses)
   - OR tap the three dots menu → Share
3. Select **"Show QR Code"**
4. Your friend opens RoomieSpend → Home Screen → "Scan QR" button
5. They scan your QR code
6. Done! They're now in your group

**Best for:** Roommates meeting in person, group gatherings

---

### Method 2: Share Invite Link (Recommended for Remote)

1. Open your group in RoomieSpend
2. Tap the "Invite Friends" card
3. Select **"Share Invite Link"**
4. Choose WhatsApp, SMS, Email, or any app
5. Your friend receives the message with invite code
6. They open RoomieSpend → "Enter Invite Code"
7. Enter the code: RMSP-XXXX
8. Done!

**Best for:** Friends in different locations, group chats

---

### Method 3: Add from Contacts

1. Open your group in RoomieSpend
2. Tap the "Invite Friends" card
3. Select **"Add from Contacts"**
4. Grant contacts permission (one-time)
5. Search and select friends
6. Tap "Send Invite"
7. Choose how to send (SMS/WhatsApp)

**Best for:** Inviting multiple people at once

---

## How to Join a Group

### Option A: Scan QR Code
1. Open RoomieSpend
2. On Home Screen → Tap **"Scan QR"** in "Add Your Friends First" card
   - OR tap Groups tab → Scan icon
3. Point camera at friend's QR code
4. Automatic join!

### Option B: Enter Invite Code
1. Open RoomieSpend
2. On Home Screen → Tap **"Enter Code"** in "Add Your Friends First" card
   - OR tap Groups tab → Enter Code
3. Type the code (e.g., RMSP-A5B2)
4. Tap "Join Group"
5. Success!

---

## Invite Code Format

All invite codes follow this format:
- **RMSP-XXXX**
- Example: `RMSP-A5B2`
- Case-insensitive (RMSP-a5b2 works too)

---

## Important Notes

### Invite Expiration
- Invite codes expire after **7 days**
- After expiration, group admin can generate a new code
- QR codes also use the same expiring invite code

### One-Time Use
- Each invite code can be used by multiple people
- But once you've joined a group, you cannot join again with the same code

### Permissions Required
- **Camera**: For scanning QR codes
- **Contacts**: For inviting from contacts (optional)
- Both permissions are requested at the time of use

### What Happens When You Join?
1. ✅ Added to group members list
2. ✅ Can view all group expenses
3. ✅ Can add new expenses
4. ✅ Activity logged: "{Your Name} joined the group"
5. ✅ Group owner gets notified (future feature)

## Troubleshooting

### "Invalid invite code"
- Check the code format (RMSP-XXXX)
- Ask your friend for a new code
- Ensure no extra spaces

### "Invite code expired"
- Ask the group owner to generate a new invite
- They can do this from Group Details → Share

### "You are already a member"
- You're already in this group!
- Check your groups list

### "Cannot join archived group"
- This group is no longer active
- Ask the owner to unarchive it first

### "Camera permission denied"
- Go to Settings → Apps → RoomieSpend → Permissions
- Enable Camera permission
- Return to app and try again

### "Contacts permission denied"
- You can still share via link!
- Or go to Settings → Apps → RoomieSpend → Permissions
- Enable Contacts permission

## Tips & Best Practices

### For Group Owners:
1. **Generate invite code immediately** after creating a group
2. **Share code in group chat** so everyone has it
3. **Screenshot QR code** and save it to share later
4. **Set group icon** before inviting (makes it recognizable)

### For Members:
1. **Save invite code** if you want to share it with others
2. **Join quickly** before code expires
3. **Check group name** before joining to ensure it's the right one

### For Everyone:
- 🎯 **QR Code** = Fast for in-person
- 💬 **Share Link** = Best for WhatsApp/Telegram groups
- 📱 **Contacts** = Bulk invites to phone contacts

## Privacy & Security

✅ **Your contacts are safe**
- We only read contacts when you explicitly tap "Add from Contacts"
- We never upload your contacts to our servers
- Contacts permission is optional

✅ **Invite codes are secure**
- Generated using cryptographic random
- Automatically expire after 7 days
- Cannot be guessed

✅ **Control who joins**
- Only people with the invite code can join
- You can see all group members
- Group owner can remove members (future feature)
